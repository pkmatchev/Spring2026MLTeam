The ann_model, train_ann, cnn_train, and compare_results files
are all written for you. 

Your goal is to complete the cnn_model.py file!

Once completed,

cd into ./Lab2/ann and run python train_ann.py
You may have to run python3 train_ann.py

Then, cd into ./Lab2/acnn and run python train_cnn.py
You may have to run python3 train_cnn.py

Then, cd into ./Lab2 and run compare_results.py

This should generate a graph that compares the performance of the two models

Please reach out to us if you have any questions, and use StackOverflow
or python documentation for syntax help. The logic is pretty much layed out, however!
